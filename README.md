## 📋 О проекте

**CertKeeper** — это система централизованного мониторинга срока действия сертификатов, используемых в УТМ ЕГАИС и ФНС. 

Система автоматически собирает информацию о сертификатах со всех компьютеров с установленными агентами, выполняет дедупликацию (оставляет только самый свежий сертификат для каждого ИНН/организации) и предоставляет удобный веб-интерфейс для контроля.

### 🎯 Зачем это нужно?

- ⏰ **Контроль сроков** — никогда не пропустите истечение сертификата
- 📊 **Централизация** — все сертификаты со всех ПК в одном месте
- 🔍 **Дедупликация** — старые сертификаты не захламляют отчёт
- 🚨 **Оповещения** — своевременное предупреждение о скором истечении (В ПЛАНАХ)
- 📈 **Статистика** — полная картина по всем компьютерам

## ✨ Возможности

### Со стороны агента (Windows)
- ✅ Автоматическое обнаружение токенов через OpenSC
- ✅ Чтение сертификатов с Рутокен (ГОСТ для ЕГАИС)
- ✅ Поиск ФНС сертификатов в реестре Windows
- ✅ Поиск ФНС сертификатов на всех подключенных токенах
- ✅ Дедупликация на стороне агента (оставляем только свежие)
- ✅ Работа через планировщик задач (3 раза в день)
- ✅ Автосохранение отчёта локально при проблемах с сетью

### Со стороны сервера (Linux/Windows)
- ✅ Веб-интерфейс для мониторинга
- ✅ REST API для сбора данных
- ✅ Автоматическая статистика по всем компьютерам
- ✅ Разделение на проблемные/исправные ПК
- ✅ Детальная информация по каждому сертификату
- ✅ Поддержка UTF-8 (русские имена корректно отображаются)
- ✅ Готовность к запуску в Docker

## 📦 Установка и настройка

### Серверная часть (Docker)

```bash
# Клонируем репозиторий
git clone https://git.fofiki.ru/megafofik/CertKeeper.git
cd certkeeper

# Запускаем в Docker
docker-compose up -d

#Сервер будет доступен по адресу: http://ВАШ_IP:5000
```

### Клиентская часть (Windows)

- Установите OpenSC https://github.com/OpenSC/OpenSC/releases
- Установите драйверы Рутокен https://www.rutoken.ru/support/download/windows/
- Настройте адрес сервера в скрипте install_agent_scheduler.ps1 и запустите его
